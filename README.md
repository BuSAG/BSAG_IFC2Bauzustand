# BSAG_IFC2Bauzustand
Eigenentwickeltes Tool zur modellbasierten Visualisierung von Bauzuständen in IFC-Modellen. Basierend auf den Eigenschaften Bauphase und Rueckbauphase werden Bauphasen regelbasiert dargestellt, um Abläufe zu analysieren, Mengen abzuleiten und Bauphasenpläne effizient zu erstellen und zu kommunizieren.
